# Yii2 etcd component

https://etcd.io

## Required

- PHP: >= 8.1
- grpc - for RPC
- protobuf - for RPC

## Install

```bash
composer require s1lver/yii2-etcd "^1.0.0"
```

or add

```
"s1lver/yii2-etcd": "^1.0.0"
```

to the require section of your composer.json file.

## Supported etcd API version
